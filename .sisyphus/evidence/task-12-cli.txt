Sync lyrics with timestamps for a song

positional arguments:
  query          YouTube URL, Spotify URL, or title/artist search query

options:
  -h, --help     show this help message and exit
  --verbose, -v  Enable verbose logging

## Key Implementation Details
1. **URL Detection**: Checks for "youtube.com", "youtu.be", "spotify.com", "spotify:" prefixes
2. **Query Parsing**: Splits on " - " (space-dash-space) for "Artist - Title" format
3. **Error Handling**: 
   - Empty query → exit code 1 with error on stderr
   - ValueError/RuntimeError → exit code 1 with error message
   - Ctrl+C → exit code 130
4. **Output**: JSON to stdout (via `model_dump_json(indent=2)`), logs to stderr
5. **Logging**: Configurable via --verbose/-v flag (DEBUG vs WARNING level)
